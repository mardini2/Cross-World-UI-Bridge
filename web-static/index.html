What you get

A small local agent (FastAPI) + a friendly CLI to drive Chrome, Edge (CDP), Spotify, Microsoft Word (COM), and basic window focus.

  • Local API (token-protected)
  • Browser control via Edge/Chrome DevTools
  • Spotify PKCE OAuth, play/pause/now
  • Word word-count via COM (Windows)
  • Windows UI list & focus windows

Quick Start (Dev)

py -3.11 -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install --upgrade setuptools wheel
python -m pip install -r requirements.txt

uvicorn app.main:app --host 127.0.0.1 --port 5025 --reload

# new terminal
python -m app.cli.cli doctor
python -m app.cli.cli ping

Common Commands

# Browser (Edge via CDP)
python -m app.cli.cli browser launch
python -m app.cli.cli browser open "https://example.com"
python -m app.cli.cli browser tabs

# Spotify (per-user Client ID; no secrets in repo)
python -m app.cli.cli spotify config set-client-id "YOUR_CLIENT_ID"
python -m app.cli.cli spotify login
python -m app.cli.cli spotify play "a moment apart odesza"
python -m app.cli.cli spotify now
python -m app.cli.cli spotify pause

# Word COM
python -m app.cli.cli word count "C:\Path\To\Your.docx"

# Windows UI
python -m app.cli.cli window list
python -m app.cli.cli window focus "Notepad"
