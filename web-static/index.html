What you get

A small local agent (FastAPI) + a friendly CLI that drives Edge/Chrome (CDP), Spotify, Microsoft Word (COM), and window focus.

  • Local API (token-protected)
  • Browser control via DevTools
  • Spotify PKCE OAuth, play/pause/now
  • Word word-count via COM (Windows)
  • Windows UI list & focus windows

Quick Start (Dev)

py -3.11 -m venv .venv
.\.venv\Scripts\Activate.ps1\
python -m pip install --upgrade pip
python -m pip install --upgrade setuptools wheel
python -m pip install -r requirements.txt

uvicorn app.main:app --host 127.0.0.1 --port 5025 --reload

# new terminal
ui doctor
ui ping

Common Commands

# Browser (Edge via CDP)
ui browser launch
ui browser open "https://example.com"
ui browser tabs

# Spotify (per-user Client ID; no secrets in repo)
ui spotify config set-client-id "YOUR_CLIENT_ID"
ui spotify login
ui spotify play "a moment apart odesza"
ui spotify now
ui spotify pause

# Word COM
ui word count "C:\Path\To\Your.docx"

# Windows UI
ui window list
ui window focus "Notepad"

CI & Site
