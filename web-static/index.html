Cross-World UI Bridge
Control your Windows apps from a clean CLI. Local, secure, friendly.

Download

Build the installer via the included scripts. For now, run from source while you iterate.

GitHub (coming soon)

Quick Start

python -m venv .venv
. .venv/Scripts/activate
pip install -r requirements.txt
uvicorn app.main:app --host 127.0.0.1 --port 5025 --reload

Commands

ui doctor
ui ping
ui browser launch
ui browser open https://example.com
ui spotify login
ui spotify play "lofi"
ui word count "C:\path\to\doc.docx"
ui window list
ui window focus "Notepad"
